## 防抖debounce | 生/熟/秒: 
```js
function debounce(fn, delay) {
  let timer; // 定义一个定时器变量，用于存储 setTimeout 的 ID

  return function (...args) { // 返回一个新函数，事件触发时调用这个函数
    clearTimeout(timer);     // 如果上一次定时器还没执行，取消它
    timer = setTimeout(() => { // 设置一个新的定时器
      fn.apply(this, args);  // 延迟执行 fn，保证 this 和参数正确
    }, delay);
  }
}
```
- fn：你想要防抖的函数（比如搜索函数）
关键点：延迟执行、闭包保存定时器、this和参数透传。
秒写技巧：记住三步：timer → clearTimeout → setTimeout(fn, delay)。
- 闭包
debounce 执行时，timer 变量 属于 debounce 的作用域
返回的函数里访问 timer，形成闭包
所以每次调用返回的函数时，都会访问同一个 timer，保证你可以 清除之前的定时器

这就是防抖的核心：保存上一次延迟的状态
- apply
setTimeout 里的函数是 箭头函数
箭头函数不会有自己的 this，它的 this 继承自外层函数
外层函数就是 return function (...args) {...}
所以这里的 this 指向 调用防抖函数的对象

function throttle(fn, interval) {
  let last = 0;
  return function (...args) {
    const now = Date.now();
    if (now - last >= interval) {
      last = now;
      fn.apply(this, args);
    }
  }
}
```

## 深拷贝 | 生/熟/秒: 
function deepClone(obj, map = new WeakMap()) {
  if (obj === null || typeof obj !== 'object') return obj;
  if (map.has(obj)) return map.get(obj);
  let clone = Array.isArray(obj) ? [] : {};
  map.set(obj, clone);
  for (let key in obj) {
    if (obj.hasOwnProperty(key)) {
      clone[key] = deepClone(obj[key], map);
    }
  }
  return clone;
}
```
关键点：递归、处理数组/对象、循环引用。
秒写技巧：记住“三步走”：判断 → 创建 → 递归赋值
## Promise完整实现 | 生/熟/秒:
核心思路：

状态：pending/fulfilled/rejected
.then链式调用
异步执行回调
错误捕获
```js
class MyPromise {
  constructor(executor) {
    this.status = 'pending';
    this.value = undefined;
    this.callbacks = [];
    const resolve = (value) => {
      if (this.status !== 'pending') return;
      this.status = 'fulfilled';
      this.value = value;
      setTimeout(() => this.callbacks.forEach(cb => cb.onFulfilled(value)));
    }
    const reject = (reason) => {
      if (this.status !== 'pending') return;
      this.status = 'rejected';
      this.value = reason;
      setTimeout(() => this.callbacks.forEach(cb => cb.onRejected(reason)));
    }
    try { executor(resolve, reject) } catch(e) { reject(e) }
  }

  then(onFulfilled, onRejected) {
    return new MyPromise((resolve, reject) => {
      const handle = {
        onFulfilled: val => {
          try { resolve(onFulfilled(val)) } catch(e) { reject(e) }
        },
        onRejected: err => {
          try { resolve(onRejected(err)) } catch(e) { reject(e) }
        }
      }
      if (this.status === 'pending') this.callbacks.push(handle);
      else if (this.status === 'fulfilled') handle.onFulfilled(this.value);
      else handle.onRejected(this.value);
    });
  }
}
```

## 数组方法实现（map/filter/reduce） | 生/熟/秒: 
```js
Array.prototype.myMap = function(fn) {
  let res = [];
  for (let i = 0; i < this.length; i++) res.push(fn(this[i], i, this));
  return res;
}

Array.prototype.myFilter = function(fn) {
  let res = [];
  for (let i = 0; i < this.length; i++) if(fn(this[i], i, this)) res.push(this[i]);
  return res;
}

Array.prototype.myReduce = function(fn, init) {
  let acc = init === undefined ? this[0] : init;
  let start = init === undefined ? 1 : 0;
  for (let i = start; i < this.length; i++) acc = fn(acc, this[i], i, this);
  return acc;
}
```
关键点：循环、回调、累积值处理。
秒写技巧：思路固定 → push/acc。

## call/apply/bind | 生/熟/秒: 
```js
Function.prototype.myCall = function(ctx, ...args) {
  ctx = ctx || globalThis;
  const fnSymbol = Symbol();
  ctx[fnSymbol] = this;
  const res = ctx[fnSymbol](...args);
  delete ctx[fnSymbol];
  return res;
}

Function.prototype.myApply = function(ctx, args) {
  return this.myCall(ctx, ...(args||[]));
}

Function.prototype.myBind = function(ctx, ...args) {
  const fn = this;
  return function(...newArgs) {
    return fn.apply(this instanceof fn ? this : ctx, [...args, ...newArgs]);
  }
}
```
关键点：this绑定、参数透传、构造函数兼容。
秒写技巧：先Call，再改Apply和Bind。

## new/instanceof | 生/熟/秒: 
```js
function myNew(constructor, ...args) {
  let obj = {};
  obj.__proto__ = constructor.prototype;
  const res = constructor.apply(obj, args);
  return typeof res === 'object' && res !== null ? res : obj;
}

function myInstanceof(obj, constructor) {
  let proto = obj.__proto__;
  while(proto) {
    if(proto === constructor.prototype) return true;
    proto = proto.__proto__;
  }
  return false;
}
```
关键点：原型链、构造函数执行。
秒写技巧：记住“proto 指向 prototype”。
