## Markdown常用语法速查表

| 用途 | 语法 | 效果 |
|------|------|------|
| **一级标题** | `# 标题` | <h1>标题</h1> |
| **加粗** | `**加粗**` 或 `__加粗__` | **加粗** |
| **斜体** | `*斜体*` 或 `_斜体_` | *斜体* |
| **粗斜体** | `***粗斜体***` | ***粗斜体*** |
| **删除线** | `~~删除线~~` | ~~删除线~~ |
| **无序列表** | `- 项目` 或 `* 项目` | • 项目 |
| **有序列表** | `1. 项目` | 1. 项目 |
| **任务列表** | `- [ ] 任务`<br>`- [x] 完成` | - [ ] 任务<br>- [x] 完成 |
| **行内代码** | `` `代码` `` | `console.log()` |
| **代码块** | ```` ```语言```` | 带高亮的代码块 |
| **链接** | `[文本](链接)` | [百度](https://baidu.com) |
| **图片** | `![描述](图片链接)` | 显示图片 |
| **引用** | `> 引用内容` | > 引用内容 |
| **分割线** | `---` 或 `***` | --- |
| **表格** | `\|列1\|列2\|` | 表格 |
- 想控制图片大小：
```js
<img src="logo.png" width="150" alt="项目Logo">
```
## 代码块示例
| 语言 | 语法 | 用途 |
|------|------|------|
| JavaScript | ```` ```javascript ```` | JS代码高亮 |
| HTML | ```` ```html ```` | HTML代码 |
| CSS | ```` ```css ```` | 样式代码 |
| Bash/Shell | ```` ```bash ```` | 命令代码 |
| JSON | ```` ```json ```` | 配置文件 |
| Markdown | ```` ```markdown ```` | Markdown自身 |

## 表格详细语法
```markdown
| 表头1 | 表头2 | 表头3 |
|-------|-------|-------|
| 内容1 | 内容2 | 内容3 |
| 左对齐 | 居中对齐 | 右对齐 |
```

对齐控制：
- `:---` 左对齐
- `:---:` 居中对齐
- `---:` 右对齐
